# 🎨 AI Sketch Mentor

An AI powered creative coach that helps artists improve through structured feedback, progress tracking, and personalized learning paths.

---

## 🧠 What it does

AI Sketch Mentor reviews user sketches and gives detailed feedback across:

- Anatomy
- Shading
- Perspective
- Composition
- Line confidence

Instead of generic advice, it behaves like a real mentor that explains what to fix and how to improve.

It also tracks progress over time so users can see real improvement, not just isolated feedback.

---

## 🎯 Why it exists

Most art feedback tools stop at surface level critique.

This system is designed to feel like a real learning experience where:

- feedback is structured
- progress is measurable
- improvement is visible over time
- users build consistency and momentum

---

## ⚡ Key features

- AI powered sketch analysis using vision models
- Structured critique with actionable improvement steps
- Skill progression tracking per user
- Daily creative challenges
- Streak based motivation system
- Secure authentication and saved history
